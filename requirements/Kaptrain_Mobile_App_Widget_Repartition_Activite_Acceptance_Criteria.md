
## APP – FR-XXX Répartition d’activité (Widget)

**Description**  
Allow users to visualize the distribution of their physical activities over a selected period with multiple views and sorting options.

**Actor**  
Athlete

**Priority**  
Medium

**Acceptance Criteria**
- User can access the “Répartition d’activité” widget from statistics.
- User can switch between time ranges: Week, Month, Year.
- User can navigate between periods using previous/next controls.
- System displays activity distribution as a donut chart.
- User can switch distribution mode:
  - Time spent (hours)
  - Number of sessions
  - RPE Cognitive
  - RPE Physical
- Chart updates dynamically when distribution mode changes.
- User can open color‑code settings.
- User can assign a custom color to each sport.
- “Other sports” category is displayed when applicable.
- User can save color customizations.
- Detailed breakdown (time, percentage) is displayed per sport.
- If no data exists for the selected period, an empty‑state message is shown.
