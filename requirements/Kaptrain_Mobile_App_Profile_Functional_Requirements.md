# Kaptrain Mobile App – Profile Functional Requirements

## APP – FR-PROFILE-OVERVIEW
**Description**  
Provide the user with a centralized Profile screen allowing access to personal training data, preferences, and shortcuts to key profile-related features.

**Actor**  
Athlete (Mobile App User)

**Priority**  
High

**Acceptance Criteria**
- The Profile screen is accessible from the bottom navigation bar.
- The user’s avatar, name, and role are displayed at the top of the screen.
- The Profile screen summarizes key information and provides navigation to detailed sections.

---

## APP – FR-PROFILE-WELLNESS-SUMMARY
**Description**  
Display a summary of the user’s wellness and training form status directly on the Profile screen.

**Actor**  
Athlete

**Priority**  
High

**Acceptance Criteria**
- The system displays the current wellness/form status (e.g., icons or indicators).
- The current streak (number of consecutive days tracked) is visible.
- The user can clearly understand their recent training consistency.

---

## APP – FR-PROFILE-MY-SPORTS
**Description**  
Allow the user to view and manage their selected sports preferences.

**Actor**  
Athlete

**Priority**  
Medium

**Acceptance Criteria**
- A “Mes sports” entry is visible in the Profile menu.
- The user can access a detailed view listing their selected sports.
- The system reflects the sports chosen during onboarding or later updates.

---

## APP – FR-PROFILE-MY-COACH
**Description**  
Provide access to information about the user’s assigned coach.

**Actor**  
Athlete

**Priority**  
Medium

**Acceptance Criteria**
- A “Mon coach” entry is visible in the Profile menu.
- The user can view coach details (name, avatar, role).
- If no coach is assigned, the system displays an appropriate empty or invitation state.

---

## APP – FR-PROFILE-MY-RECORDS
**Description**  
Allow the user to consult their personal performance records.

**Actor**  
Athlete

**Priority**  
Medium

**Acceptance Criteria**
- A “Mes records” entry is visible in the Profile menu.
- The user can access a list of personal bests or tracked achievements.
- Records are displayed in a clear and readable format.

---

## APP – FR-PROFILE-MY-INJURIES
**Description**  
Allow the user to view and manage declared injuries or physical limitations.

**Actor**  
Athlete

**Priority**  
Medium

**Acceptance Criteria**
- A “Mes blessures” entry is visible in the Profile menu.
- The user can view existing injury information.
- The system supports an empty state if no injuries are declared.

---

## APP – FR-PROFILE-PHYSIOLOGICAL-DATA
**Description**  
Provide access to physiological data relevant to training personalization.

**Actor**  
Athlete

**Priority**  
Low

**Acceptance Criteria**
- A “Mes données physiologiques” entry is visible in the Profile menu.
- The user can consult stored physiological metrics.
- Data is presented in a structured and understandable manner.

---

## APP – FR-PROFILE-SETTINGS-ACCESS
**Description**  
Allow the user to access profile-related settings from the Profile screen.

**Actor**  
Athlete

**Priority**  
Medium

**Acceptance Criteria**
- A settings icon is displayed on the Profile screen.
- Tapping the icon navigates the user to profile and application settings.
- Settings access does not interrupt ongoing tracking or sessions.
