## APP – FR-AGD-080 Types de séances

**Description**  
Allow users to view, execute, and complete different types of training sessions depending on their origin: individual, personal, programme, or programmation.

**Actor**  
Athlete

**Priority**  
High

**Acceptance Criteria**
- User can access a session from the Agenda regardless of its type.
- System displays the session type clearly (Individualisée, Perso, Programme, Programmation).
- Each session displays structured blocks (Échauffement, Travail principal, Récupération).
- User can expand or collapse each block.
- Exercises associated with each block are displayed with video previews.
- User can add a personal note to the session.
- User can mark a block as completed.
- Completion status is visually updated in real time.
- User can mark the entire session as “J’ai terminé”.
- Completed sessions display effort indicators (RPE Physique & Cognitif) when applicable.
- For programme sessions, system displays programme name, week, and session index.
- Navigation back to Agenda preserves session completion state.
