
# Kaptrain Mobile App — Paramètres (Settings)
ID: APP-FR-XXX

## Overview
Settings allow users to manage profile, preferences, notifications, security, and legal information.

## Profile Settings
- Edit profile information
- Change avatar (crop & resize)
- Change gender
- Update height, weight, practice level
- Delete account (with confirmation modal)

## Security
- Change password (old, new, confirm)
- Logout with confirmation

## Training Preferences
- Configure wellness tracking behavior
- Enable/disable daily wellness reminder
- Set default wellness prompt behavior

## Application Preferences
- Choose units:
  - Weight: kg / lbs
  - Height: cm / ft-in
- Language:
  - French
  - English

## Notifications
- Toggle training notifications on/off

## Legal & Support
- Conditions d’utilisation
- Politique de confidentialité
- Contact support:
  - Phone
  - Email

## Functional Requirements
- All changes require explicit save
- Modals for destructive actions
- Persist preferences per user

## Non-Functional
- Accessibility compliant
- Immediate UI feedback on save
