
## APP – FR-XXX Open Exercise Filters
**Description** Allow Athletes to open the filters panel from the Exercise Library.
**Actor** Athlete
**Priority** High
**Acceptance Criteria**
- Athlete can open the Filters screen from the Exercise Library.
- The Filters screen displays categories: Sports, Thématiques, Matériels, Muscles.
- Athlete can navigate back to the Exercise Library without applying filters.

---

## APP – FR-XXX Filter Exercises by Sports
**Description** Allow Athletes to filter exercises based on selected sports.
**Actor** Athlete
**Priority** High
**Acceptance Criteria**
- Athlete can view a list of available sports.
- Athlete can select one or multiple sports.
- Selected sports are visually highlighted.
- Applying the filter updates the exercise list accordingly.
- Athlete can clear selected sports using “Tout effacer”.

---

## APP – FR-XXX Filter Exercises by Thématiques
**Description** Allow Athletes to filter exercises by training thematics and body zones.
**Actor** Athlete
**Priority** Medium
**Acceptance Criteria**
- Athlete can select one or more thematics.
- Athlete can select body zones (e.g. Membres inférieurs, Core).
- Athlete can apply or reset the selected thematics.
- Exercise results update based on selected thematics.

---

## APP – FR-XXX Filter Exercises by Matériels
**Description** Allow Athletes to filter exercises based on required equipment.
**Actor** Athlete
**Priority** Medium
**Acceptance Criteria**
- Athlete can view a grid of available equipment.
- Athlete can select “Sans matériel” or multiple equipment types.
- Selected equipment is clearly indicated.
- Applying the filter refreshes the exercise list.

---

## APP – FR-XXX Filter Exercises by Muscles (List & Body Map)
**Description** Allow Athletes to filter exercises by targeted muscles using list or body map.
**Actor** Athlete
**Priority** Medium
**Acceptance Criteria**
- Athlete can switch between List view and Corps humain view.
- Athlete can select one or multiple muscles.
- Selected muscles are highlighted in both views.
- Applying the filter updates exercises based on targeted muscles.

---

## APP – FR-XXX Reset Exercise Filters
**Description** Allow Athletes to reset all applied exercise filters.
**Actor** Athlete
**Priority** Low
**Acceptance Criteria**
- Athlete can reset all filters using “Tout effacer”.
- All selected filters are cleared.
- Exercise list returns to the unfiltered state.
