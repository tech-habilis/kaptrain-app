## APP – FR-AG-Notes-01 Historique de notes

**Description**  
Allow users to view the history of personal notes associated with their training sessions, grouped by sport and session date.

**Actor**  
Athlete, Coach

**Priority**  
Medium

**Acceptance Criteria**
- User can access the “Historique de mes notes” screen from the agenda or session context.
- Notes are grouped by sport.
- Each sport section can be expanded or collapsed.
- Each session displays its date.
- Notes are grouped by training block (e.g. Échauffement, Étirements).
- Each note shows its content and creation date.
- If no notes exist for a session, the session is still listed without notes.
- Scrolling allows viewing older notes.
- Back navigation returns the user to the previous agenda screen.
