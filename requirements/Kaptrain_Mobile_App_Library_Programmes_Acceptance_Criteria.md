## APP – FR-XXX View Programmes Library
**Description**  
Allow users to browse all available training programmes in the Library.

**Actor**  
User

**Priority**  
High

**Acceptance Criteria**  
- User can access the Programmes section from the Library tab.
- The system displays a list of available programmes.
- Each programme shows key information (title, level, duration, frequency).
- User can scroll through the list of programmes.
- User can select a programme to view its details.


## APP – FR-XXX View Purchased Programmes
**Description**  
Allow users to view programmes they have already purchased.

**Actor**  
User

**Priority**  
High

**Acceptance Criteria**  
- User can view a “Mes achats” section.
- The system displays only programmes purchased by the user.
- Each purchased programme shows its current status (not started, in progress, completed).
- User can access a purchased programme’s detail page.


## APP – FR-XXX View Programme Details
**Description**  
Allow users to view detailed information about a training programme.

**Actor**  
User

**Priority**  
High

**Acceptance Criteria**  
- User can open a programme detail screen.
- The system displays programme description, level, duration, and session frequency.
- The system displays programme objectives.
- The system displays required equipment (if applicable).
- The system displays the coach or programme author.


## APP – FR-XXX Purchase a Programme
**Description**  
Allow users to purchase a paid training programme.

**Actor**  
User

**Priority**  
High

**Acceptance Criteria**  
- User can initiate the purchase of a programme.
- The system displays the programme price clearly.
- User can select a payment method.
- The system validates payment information before confirmation.
- Upon successful payment, the programme is added to the user’s purchased programmes.


## APP – FR-XXX Confirm Programme Payment
**Description**  
Confirm a successful programme payment and provide next actions.

**Actor**  
User

**Priority**  
High

**Acceptance Criteria**  
- The system displays a payment confirmation screen.
- The system confirms that the programme is unlocked.
- User can navigate back to the home screen.
- User can navigate directly to the purchased programme.


## APP – FR-XXX Start a Programme
**Description**  
Allow users to start a purchased programme.

**Actor**  
User

**Priority**  
Medium

**Acceptance Criteria**  
- User can start a purchased programme.
- The system marks the programme as “in progress”.
- The system schedules programme sessions in the user’s agenda.
- The system allows the user to resume the programme later.


## APP – FR-XXX Restart a Programme
**Description**  
Allow users to restart a previously completed programme.

**Actor**  
User

**Priority**  
Low

**Acceptance Criteria**  
- User can restart a completed programme.
- The system resets programme progress.
- The system updates the agenda with new sessions.
- Previous completion history remains accessible.


## APP – FR-XXX Access Programme from Agenda
**Description**  
Allow users to access programme sessions directly from the Agenda.

**Actor**  
User

**Priority**  
Medium

**Acceptance Criteria**  
- Programme sessions appear in the Agenda.
- User can open a programme session from the Agenda.
- The system links the session to the correct programme.
- Session status updates correctly after completion.
