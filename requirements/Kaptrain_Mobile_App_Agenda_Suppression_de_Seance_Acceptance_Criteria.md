## APP – FR-AG-008 Suppression de séance

**Description**  
Allow users to delete an existing training session from the agenda.

**Actor**  
Athlete, Coach

**Priority**  
High

**Acceptance Criteria**
- User can initiate session deletion from the session edit screen.
- A delete icon is visible on each training block and at session level.
- System displays a confirmation modal before deletion.
- Confirmation modal clearly indicates that deletion is irreversible.
- User can cancel deletion and return to the edit screen.
- Upon confirmation, the session is removed from the agenda.
- Deleted session no longer appears in daily, weekly, or monthly views.
- System updates agenda data immediately after deletion.
