# Kaptrain Mobile App – Mon Coach (Functional Requirements)

## APP-FR-XXX – Mon Coach

### Overview
The **Mon Coach** feature allows an athlete to view their current coach and optionally connect to a coach using an invitation code.

### Core Features
- Display current coach information
- Invitation code input
- Coach connection workflow

### User Interface
- Coach card with avatar, name, and start date
- Invitation section with input field
- Primary action button to connect
- Floating action button for quick access

### Functional Requirements
- Show current coach if already connected
- Allow input of invitation code
- Validate invitation code before connection
- Show error feedback for invalid codes
- Update coach association upon success

### Edge Cases
- Athlete without coach
- Invalid or expired invitation code
- Network failure during connection

### Non-Functional Requirements
- Secure coach-athlete association
- Instant UI feedback
- Consistent behavior across platforms
