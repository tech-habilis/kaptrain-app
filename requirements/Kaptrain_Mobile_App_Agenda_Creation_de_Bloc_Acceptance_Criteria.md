## APP – FR-201 Création de bloc de séance

**Description**  
Allow Coaches to create and configure a training block within a session, including intensity reference, exercises, and parameters.

**Actor**  
Coach

**Priority**  
High

**Acceptance Criteria**
- Coach can add a new block from the session creation flow.
- Coach can view the block title, description, and training objective.
- Coach can select an intensity reference (e.g. %RM, %FC Max, %PMA, %FTP, %VMA, RPE).
- If an intensity reference is selected, related configuration fields are displayed.
- Coach can configure intensity using time or distance depending on the reference type.
- Coach can select a training zone and percentage range when applicable.
- Coach can view reference tables (e.g. VMA zones and target speeds).
- Coach can add one or more exercises to the block.
- Coach can search and filter exercises when associating them to the block.
- Coach can associate exercise videos to the block.
- Coach can save the block and return to the session overview.
- Saved blocks are displayed in the session with their configured exercises and intensity.
