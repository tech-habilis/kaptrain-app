## APP – FR-AGENDA-END-SESSION Fin de séance

**Description**  
Allow users to complete a training session by entering duration and subjective effort feedback (RPE), in order to enrich training analysis and history.

**Actor**  
Athlete

**Priority**  
High

**Acceptance Criteria**
- User is prompted to complete the end-of-session flow after finishing a session or timer.
- User can input the actual duration of the session using a circular selector.
- User can proceed to the next step after confirming the session duration.
- User can evaluate the session using a general perceived effort slider.
- User can optionally provide written feedback about how the session felt.
- User can evaluate Physical RPE (RPE Physique) on a scale from no effort to maximal effort.
- User can evaluate Cognitive RPE (RPE Cognitif) on a scale from no effort to maximal effort.
- User can submit the evaluation and save the session feedback.
- User can choose “Remplir plus tard” to skip the evaluation without blocking navigation.
- System stores duration and RPE data linked to the completed session.
- Completed session is marked as finished in the agenda and history.
