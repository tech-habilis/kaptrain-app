
## APP – FR-XXX Volume d’entraînement (Widget)

**Description**  
Allow users to visualize their training volume over time based on completed sessions.

**Actor**  
Athlete

**Priority**  
Medium

**Acceptance Criteria**
- User can access the “Volume d’entraînement” widget from the statistics section.
- User can switch between time ranges: Week, Month, and Year.
- The chart displays total training duration aggregated per selected period.
- Weekly view shows daily bars with duration labels.
- Monthly view shows day-by-day aggregation within the selected month.
- Yearly view shows month-by-month aggregation.
- User can navigate between periods using previous/next controls.
- Chart updates dynamically when the selected time range changes.
- If no data exists for a period, an empty-state message is displayed.
