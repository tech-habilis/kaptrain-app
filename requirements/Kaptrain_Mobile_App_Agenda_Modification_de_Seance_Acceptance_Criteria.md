
## APP – FR-AG-010 Modification de séance

**Description**  
Allow users to modify an existing training session, including its metadata, blocks, and associated exercises.

**Actor**  
Athlete, Coach

**Priority**  
High

**Acceptance Criteria**
- User can access the session modification screen from the Agenda.
- User can edit the session title.
- User can view the list of existing training blocks.
- User can add a new training block to the session.
- User can edit an existing training block.
- User can delete an existing training block.
- User can modify session metadata:
  - Sport / thematic
  - Date
- User can expand or collapse additional information.
- User can save changes to the session.
- System persists all modifications after saving.
- User can delete the entire session.
- System requires confirmation before deleting a session.
- After deletion, the session is removed from the Agenda.
- User is redirected back to the Agenda after successful save or deletion.
